Sobre el proyecto

Esta es una aplicación web sencilla para gestionar tareas personales. Fue desarrollada usando el framework Flask y una base de datos SQLite.

Los usuarios pueden registrarse, iniciar sesión, agregar nuevas tareas, visualizar su lista personal de pendientes y eliminarlas cuando ya estén completadas. Cada usuario tiene acceso únicamente a sus propias tareas.

Este proyecto tiene como objetivo aprender los fundamentos del desarrollo web con Flask, autenticación de usuarios y operaciones CRUD (Crear, Leer, Actualizar, Borrar) sobre una base de datos.

El código está organizado para ser claro y fácil de mantener, y puede servir como base para proyectos más complejos.
